## Prerequisites

- node &gt;= 8 &lt;11

## Getting Started

### Client

#### Install

```sh
yarn
```

#### Launch Next.js

```sh
yarn run dev:web
```

#### React-Native Android

```sh
yarn run dev:mobile
```

### Server

#### Install

```sh
yarn
```

#### Start Server

```sh
yarn run server
```

### Install packages

```
yarn workspace @onepass/<mobile/web> add <package_name>
```
